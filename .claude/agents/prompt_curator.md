```
  - Full (7/7)
  - Half (3-4/7)
```

### Adding to Golden Dataset
```
Criteria for inclusion:
  1. Perfect Merlin voice (5/5)
  2. Biome-coherent setting
  3. All 3 options meaningfully different
  4. Effects match narrative logic
  5. No anachronisms
  6. Celtic references verified against lore bible
  7. French B1 level vocabulary
  8. Diverse tag coverage

Process:
  1. Review candidate card against all 8 criteria
  2. Assign quality score (1-5)
  3. Add context metadata
  4. Write review notes
  5. Add to golden_cards.json
  6. Run regression: does existing LLM still pass?
```

## Anti-Hallucination Celtic Validation

### Verified Celtic Entities
```
Tier 1 (Core — always valid):
  Brocéliande, Avalon, Annwn, Camelot
  Merlin, Arthur, Viviane, Morgane
  Ogham, Awen, Druide, Barde, Ovate

Tier 2 (Mythology — valid with care):
  Dagda, Brigit, Cernunnos, Lugh, Morrigan
  Korrigan, Ankou, Sidhe, Tuatha De Danann
  Samhain, Beltane, Imbolc, Lughnasadh

Tier 3 (Historical — requires validation):
  Carnac, Gavrinis, Newgrange
  Vercingetorix, Boudicca
  Specific tribal names, battle references

BLACKLIST (Made-up / Hallucinated — REJECT):
  Any term not in the lore bible that sounds Celtic
  Portmanteau Celtic words (e.g., "Druicornos", "Samhainvel")
  Real-world locations used incorrectly
  Modern interpretations presented as ancient truth
```

### Validation Process
```gdscript
# Step 1: Extract Celtic-sounding terms from LLM output
func extract_celtic_terms(text: String) -> Array[String]:
    var celtic_regex := RegEx.new()
    celtic_regex.compile("(?i)(ogham|awen|druide|sidhe|ankou|korrigan|samhain|" +
        "beltane|imbolc|lughnasadh|brocéliande|avalon|annwn|camelot|" +
        "dagda|brigit|cernunnos|morrigan|tuatha)")
    var terms := []
    for result in celtic_regex.search_all(text):
        terms.append(result.get_string())
    return terms

# Step 2: Check against lore bible
func validate_celtic_reference(term: String) -> bool:
    return term.to_lower() in VERIFIED_CELTIC_ENTITIES

# Step 3: Flag unknown Celtic-sounding terms
func flag_unknown_celtic(text: String) -> Array[String]:
    var suspicious_regex := RegEx.new()
    suspicious_regex.compile("(?i)([A-Z][a-z]+(?:wn|dh|gh|th|ch|rn|nn|ll|dd|ff))")
    var flagged := []
    for result in suspicious_regex.search_all(text):
        var term := result.get_string()
        if not validate_celtic_reference(term):
            flagged.append(term)
    return flagged
```

### Template Lifecycle
```
1. DRAFT: New template written
2. TEST: Evaluated against golden dataset (20+ cards)
3. STAGED: Ready for A/B testing
4. ACTIVE: Currently used in production
5. DEPRECATED: Replaced by newer version
6. ARCHIVED: Kept for reference

Location: data/ai/config/prompt_templates.json
```

## RAG Context Quality

### Context Assessment
```
What the RAG sends to the LLM directly affects output quality.
Review RAG context for:

1. Relevance: Does context match current game state?
2. Completeness: Are critical fields present (aspects, biome, souffle)?
3. Conciseness: Is context within 180 token budget?
4. Freshness: Does context reflect recent events?
5. Priority: Are CRITICAL items always included?

RAG Context Priority Levels:
  CRITICAL (4): aspects, souffle, biome — always included
  HIGH (3): active arcs, flux tiers — usually included
  MEDIUM (2): recent choices, faction relations — space permitting
  LOW (1): session context, weather — if room
  OPTIONAL (0): flavor text — only if budget allows
```

### Context Quality Checklist
```
- [ ] Aspect states present and correct
- [ ] Souffle count present and correct
- [ ] Biome matches current scene
- [ ] Recent choices relevant (not stale)
- [ ] No contradictory information
- [ ] Token count within budget (180)
- [ ] Priority items not displaced by low-priority
```

## Cultural Sensitivity

### Guidelines
```
Respect:
  - Celtic traditions are living cultures (Irish, Welsh, Scottish, Breton)
  - Druidism has modern practitioners
  - Ogham is a real historical writing system
  - These are not "fantasy" — they are heritage

DO:
  - Reference established mythology accurately
  - Use authentic Celtic terms (validated)
  - Present traditions with reverence
  - Acknowledge multiple interpretations
  - Cite sources when possible

DON'T:
  - Invent fake Celtic terms
  - Trivialize sacred practices
  - Mix unrelated Celtic traditions carelessly
  - Present speculation as established fact
  - Use stereotypes (leprechauns, pots of gold, etc.)
  - Appropriate sacred symbols without understanding
```

### Content Review Process
```
For any content referencing Celtic culture:
  1. Check against lore bible (primary)
  2. Cross-reference with historien_bretagne agent
  3. Verify with lore_writer agent
  4. Flag uncertain references for manual review
  5. Document validation in golden dataset notes
```
