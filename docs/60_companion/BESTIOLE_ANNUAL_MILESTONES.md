# Bestiole Annual Milestones (No Gating)

Milestones are soft goals, not requirements.
They create variety without forcing play.

Milestones (examples)
- First Bond 6: unlock first trait
- First training: unlock a form choice
- First promise kept: unlock Anchor action
- First mystery resolved: unlock Echo Trait choice
- First winter care streak (3 days): unlock Warm Nest trait
- First summer training streak (2 sessions): unlock Galeform choice
- First autumn calm event: unlock Mossform choice
- First spring harvest: unlock Seed Pouch item

Notes
- Milestones are based on actions and outcomes, not calendar dates.
- Use them as soft narrative beats, not gates.
