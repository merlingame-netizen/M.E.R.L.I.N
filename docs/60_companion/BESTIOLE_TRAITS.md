# Bestiole Traits (Echo Traits)

Traits are minor, utilitarian perks unlocked by Bond milestones.
Pick 1 trait per milestone to avoid overload.

Trait List (12)
1) Path Finder
- Effect: chance to reveal a hidden path in events
- Trigger: event with "path" tag

2) Oath Keeper
- Effect: reduce promise penalty once per cycle
- Trigger: first broken promise in cycle

3) Fog Sense
- Effect: improved success in night/mystery events
- Trigger: time_phase = night

4) Calm Breath
- Effect: reduce conflict spikes by a small amount
- Trigger: conflict increase event

5) Steady Paw
- Effect: reduce status effect chance in combat
- Trigger: when status would apply

6) Bright Eye
- Effect: improve Scout action reliability
- Trigger: Scout action used

7) Soft Pelt
- Effect: minor healing after combat
- Trigger: combat end

8) Keen Ear
- Effect: higher chance to detect trick or lie
- Trigger: trick event

9) Quiet Steps
- Effect: reduce event risk on stealth choices
- Trigger: event with stealth tag

10) Warm Nest
- Effect: energy recovery bonus after rest
- Trigger: rest action used

11) Clear Memory
- Effect: increases chance to recall hint fragments
- Trigger: mystery resolved

12) Gentle Pull
- Effect: increases Bond gain from care actions
- Trigger: any care action

Notes
- Traits should be soft bonuses, never required.
- Most traits should have a cooldown or per-day cap.
