# Bestiole Forms (Extended)

Add these to the 4 core forms to increase diversity.

Core Forms (reminder)
- Veilform: stealth, fog, scouting
- Stoneform: defense, stability, oath support
- Emberform: burst, passion, conflict control
- Riverform: heal, cleanse, memory

Additional Forms (6)
1) Thornform
- Theme: protection, path control, crowding
- Bonus: reduce event risk in forest paths

2) Galeform
- Theme: speed, reposition, initiative
- Bonus: higher chance to act first in combat

3) Rootform
- Theme: grounding, promise stability
- Bonus: reduces promise penalty magnitude

4) Tideform
- Theme: cleanse, drift, gentle pull
- Bonus: clears one minor negative effect per day

5) Emberglass
- Theme: precise burst, risk tradeoff
- Bonus: boosts damage but increases fatigue

6) Mossform
- Theme: recovery, patience, slow growth
- Bonus: increases Bond gain over time

Notes
- Forms should be reversible once per season.
- Each form should map to 2-3 actions and 1 trait synergy.
