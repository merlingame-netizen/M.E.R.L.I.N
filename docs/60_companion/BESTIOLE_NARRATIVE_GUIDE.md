# Bestiole Narrative Guide

Goal
- Integrate Bestiole into events without stealing focus.

Guidelines
- Bestiole speaks in short, ambiguous lines.
- Bestiole reacts to promises and mood shifts.
- Bestiole gives small, optional hints, never answers.

Event Hooks
- Promise success: Bestiole circles a menhir, then rests.
- Promise failure: Bestiole avoids eye contact, seeks fog.
- Mystery solved: Bestiole sniffs a path, pauses, then leads.
- Conflict drop: Bestiole relaxes, breath slows.

Tone
- Gentle, attentive, never comedic.
- No modern words.

Placement
- 1 Bestiole line per event max.
- Prefer physical reactions over dialogue.

Examples
- "Il y a un chemin qui n aime pas le bruit."
- "La pierre te reconnait."
- "Le souffle revient."
