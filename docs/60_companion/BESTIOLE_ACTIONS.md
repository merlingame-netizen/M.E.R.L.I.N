# Bestiole Actions (Combat + Utility)

Use these actions for Bestiole during combat and events.
Each round: Player action + Bestiole action.

Action List (12)
1) Guard
- Type: combat
- Effect: reduce next damage by a flat amount
- Condition: always

2) Scout
- Type: combat/event
- Effect: reveal enemy intent or hidden option
- Condition: Bond >= 2

3) Mark
- Type: combat
- Effect: increase next hit chance vs target
- Condition: Focus >= 2

4) Nudge
- Type: combat
- Effect: small heal or cleanse minor status
- Condition: Spirit >= 2

5) Ward
- Type: combat
- Effect: reduce status effect chance
- Condition: Stoneform bonus

6) Spur
- Type: combat
- Effect: boost player damage on next action
- Condition: Emberform bonus

7) Veil Step
- Type: combat/event
- Effect: dodge next attack or bypass event risk
- Condition: Veilform bonus

8) River Song
- Type: combat/event
- Effect: restore small Energy + lower Stress
- Condition: Riverform bonus

9) Fetch
- Type: event
- Effect: find a resource or clue
- Condition: Mood >= 2

10) Calm Breath
- Type: event
- Effect: reduce conflict spike or fear
- Condition: Bond >= 3

11) Anchor
- Type: event
- Effect: stabilize a promise outcome (small bonus)
- Condition: Promise active

12) Echo Call
- Type: event
- Effect: chance to trigger a rare hint or path
- Condition: Night + Spirit >= 3

Notes
- Actions should be useful, not mandatory.
- Cooldowns can be 1-3 turns for strong actions.
- Actions scale softly with Bond.
