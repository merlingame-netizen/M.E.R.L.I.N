# Bestiole UI Notes

UI Goals
- Show status without stress.
- Encourage light care actions.

HUD Elements
- Bestiole portrait + mood icon
- 4 small bars: Hunger, Energy, Mood, Cleanliness
- Bond shown as a soft ring (not a bar)
- One-line hint: "Bestiole seems calm" etc.

Interaction
- One button: Quick Care (auto chooses the best action)
- Optional expand: feed, play, groom, rest

Notifications
- Soft, rare: 1 per day max
- No red warnings, use neutral phrasing

Accessibility
- Color + icon + short text
- Avoid guilt framing
