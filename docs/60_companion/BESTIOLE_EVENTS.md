# Bestiole Events (Season + Progression)

These events are optional flavor and growth moments.
They do not gate core content.

Event Set (8)
1) Spring - First Burrow
- Trigger: first week of spring OR bond milestone
- Outcome: unlock a small trait choice

2) Summer - Heat Drift
- Trigger: 3 trainings in summer
- Outcome: temporary action bonus for 2 days

3) Autumn - Leaf Hide
- Trigger: any fog event + calm mood
- Outcome: reveal a hidden path once

4) Winter - Stone Nap
- Trigger: low energy + rest action
- Outcome: boost next day's energy gain

5) Oath Day - Silent Witness
- Trigger: promise success
- Outcome: Bond +1, unlock Anchor action

6) Oath Break - Closed Path
- Trigger: promise failure
- Outcome: mood drop, unlock special care mini event

7) Drift Echo
- Trigger: 3 mysteries resolved
- Outcome: Bestiole gains an Echo Trait choice

8) Menhir Pulse
- Trigger: conflict drop from riven to calm
- Outcome: Bestiole form attunes (temporary perk)

Notes
- Events scale by bond, not by time.
- Keep outcomes utilitarian and reversible.
