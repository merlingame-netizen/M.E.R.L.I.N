# DOC 08 — MVP profond (checklist livrable)

## MVP technique
- Store unique + phases (GameManager)
- Save slots (3) + reprise de run
- RNG seed déterministe
- EffectEngine whitelist + logs
- UI 3 boutons FORCE/LOGIQUE/FINESSE + sous-choix
- Event pipeline + 4 mini-jeux
- Combat pipeline + intentions + posture + momentum
- Map STS-like (chemin + embranchements)
- LLM Adapter JSON + validation + mémoire Merlin (I/O)

---

## MVP gameplay (contenu minimal)
- 12 events “Prison” (générables)
- 10 ennemis “Prison” (patterns)
- 30 effets whitelist
- 30 moves (dont 12 BOTH)
- 10 achievements → 5 packages loadout
- Evolution Bestiole stage 0..4 (démo) + variantes élémentaires (recolor)

---

## Critères de qualité (non frustration)
- 1 run courte doit toujours faire progresser : essence/bond/achievements
- fail streak réduit la difficulté à court terme
- UX : chance/risque/gain cohérents et lisibles
