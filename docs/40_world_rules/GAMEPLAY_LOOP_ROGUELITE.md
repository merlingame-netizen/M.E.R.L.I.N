# Roguelite Gameplay Structure — Swipe Cards (UPDATED 2026-02-05)

---

## PIVOT: De Combat vers Narration Pure

Le gameplay pivote d'un roguelite avec combat vers un **jeu de cartes narratif**:
- Pas de combat traditionnel
- Cartes narratives avec choix binaires
- 4 jauges a equilibrer (mort si 0 ou 100)
- LLM genere les scenarios dynamiquement

---

## Purpose
- Define the core game loop and how choices scale in complexity over time.
- Provide a complete reference for card types, resources, and run structure.
- Keep the game simple to play, but deep to master.

## Pillars
1. **Simple actions, complex consequences** — Swipe left/right
2. **Roguelite runs with persistent meta progression**
3. **Merlin is the overseer** — Narrator, judge, scenario generator
4. **Bestiole is passive support** — Skills (Oghams), hints, modifiers
5. **World is a controlled simulation** — Time, seasons, promises

---

## High-Level Loop (One Run)

```
1. START RUN
   - 4 jauges initialisees a 50
   - Bestiole pret avec skills equipes
   - LLM charge le contexte initial

2. CARD LOOP
   a. Merlin genere une carte (ou fallback)
   b. Afficher carte avec texte et options
   c. (Optionnel) Joueur active un skill Bestiole
   d. Joueur swipe gauche ou droite
   e. Appliquer effets aux jauges
   f. Verifier condition de fin
   g. Mettre a jour story_log et state
   h. Retour a (a)

3. END RUN
   - Trigger si n'importe quelle jauge = 0 ou 100
   - Afficher ecran de fin avec narration appropriee
   - Calculer score et unlocks
   - Sauvegarder meta progression
```

## 4 Resource Gauges

### The Core Balance
| Jauge | Theme | 0 = | 100 = |
|-------|-------|-----|-------|
| **Vigueur** | Physique | Epuisement mortel | Surmenage, burnout |
| **Esprit** | Mental/Magique | Folie, perte de soi | Possession par esprits |
| **Faveur** | Social | Exile, banni | Tyrannie, hubris |
| **Ressources** | Materiel | Famine, denument | Pillage, vol, chaos |

### Rules
- All start at **50**
- If **any** gauge hits 0 or 100 = **RUN ENDS**
- Each ending has unique narrative (8 total endings)
- Effects typically create tradeoffs (+1 somewhere, -1 elsewhere)

### Pity System
When a gauge is critical (<15 or >85):
- Increase weight of "balancing" cards
- LLM receives warning flag in context
- Bestiole hints become more obvious

## Card Types

### Narrative Cards (80%)
Generated by LLM based on current state.

```json
{
    "type": "narrative",
    "speaker": "MERLIN",
    "text": "Un voyageur s'approche de ton feu de camp...",
    "options": [
        {"direction": "left", "label": "Le chasser", "effects": [...]},
        {"direction": "right", "label": "L'accueillir", "effects": [...]}
    ],
    "tags": ["social", "stranger"]
}
```

### Event Cards (10%)
Triggered by time, season, or flags.
- Seasonal changes
- Calendar events
- Consequence of past choices

### Promise Cards (5%)
Merlin proposes a pact.
- Accept = immediate bonus + future obligation
- Refuse = no penalty, miss opportunity
- Breaking = severe consequences

### Merlin Direct (5%)
Merlin speaks directly to player.
- Hints about upcoming events
- Warnings about gauge levels
- Lore revelations

## Run Structure

### Day Cycle
Runs are divided into "days" (~20-30 cards each):
- **Matin**: More calm cards, setup opportunities
- **Apres-midi**: Peak tension, major choices
- **Soir**: Consequences and winding down
- **Nuit**: Merlin judgment, promise checks

### Season Influence
| Season | Card Weights | Effects |
|--------|--------------|---------|
| Spring | +hope, +growth | Easier recovery |
| Summer | +conflict, +passion | Higher stakes |
| Autumn | +mystery, +harvest | Balanced |
| Winter | +survival, +isolation | Slower decay |

### Promise Timing
- Promises have deadlines (in cards or days)
- Missing deadline = automatic "broken"
- Broken promise = severe gauge penalty + narrative consequence

## Bestiole Integration

### Passive Modifiers
Every card, Bestiole applies bonuses based on state:
- Mood affects effect percentages
- Bond unlocks skill slots
- Needs affect skill availability

### Active Skills (Oghams)
Player can activate ONE skill before choosing:
- **Reveal**: Show hidden effects
- **Protect**: Reduce negative effects
- **Boost**: Amplify positive effects
- **Narrative**: Add options or change cards

See: [BESTIOLE_SYSTEM.md](../60_companion/BESTIOLE_SYSTEM.md)

## Choice Complexity Ramp

### Early Run (Cards 1-30)
- Clear consequences
- Single-gauge effects
- Simple tradeoffs
- Tutorial elements

### Mid Run (Cards 31-70)
- Chained consequences (effects over multiple cards)
- Multi-gauge effects
- Promises introduced
- Callbacks to earlier choices

### Late Run (Cards 71+)
- Complex tradeoffs
- High stakes (big swings)
- Multiple active promises
- Potential for dramatic endings

## Narrative Arcs

### Arc Structure
LLM can start "arcs" that span multiple cards:
1. **Introduction**: Setup the situation
2. **Development**: Player makes choices
3. **Climax**: Major decision point
4. **Resolution**: Consequences unfold

### Arc Types
- **Character Arcs**: Recurring NPCs
- **Mystery Arcs**: Uncover secrets
- **Conflict Arcs**: Resolve tensions
- **Promise Arcs**: Fulfill or break oaths

## Meta Progression

### Persistent Across Runs
- **Gloire Points**: Earned from achievements
- **Unlocked Skills**: New Oghams for Bestiole
- **Discovered Lore**: World knowledge
- **Achievement Flags**: Track accomplishments

### Unlocks
| Gloire | Unlock |
|--------|--------|
| 10 | New starting skill |
| 25 | Extra gauge buffer (+5 starting) |
| 50 | Rare card pool access |
| 100 | New Bestiole appearance |

## Fallback System (Without LLM)

When LLM is unavailable, use pre-written card bank:

```gdscript
const FALLBACK_CARDS = [
    {
        "id": "fb_001",
        "text": "Le vent souffle fort ce soir...",
        "options": [...],
        "conditions": {"min_day": 0},
        "weight": 1.0
    },
    # 50-100 base cards
]
```

Selection based on:
- Conditions met
- Active tags
- Weight * random
- Avoid recent repeats

## Balance Targets

### Run Duration
| Type | Cards | Time |
|------|-------|------|
| Short (death) | 10-30 | 3-10 min |
| Average | 50-100 | 15-30 min |
| Long (skilled) | 150+ | 45+ min |

### Effect Ranges
| Effect Type | Typical Range |
|-------------|---------------|
| Small | +/- 5 |
| Medium | +/- 10-15 |
| Large | +/- 20-30 |
| Extreme | +/- 40+ (rare) |

### Card Distribution
- 90% tradeoff cards (+ somewhere, - elsewhere)
- 5% purely positive (rare)
- 5% purely negative (rare)
