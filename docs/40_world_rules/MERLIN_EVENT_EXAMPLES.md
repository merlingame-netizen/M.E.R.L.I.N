# Merlin Event Examples (Same Type, Different Story)

Event type: Discovery node

Example A
Hook: "La brume se fend comme un rideau mouille."
Reveal: "Un eclat d'ogham flotte au-dessus d'un puits sec."
Stakes: Gain a lore fragment, or risk a time penalty.
Choices:
1) Descendre au puits (cost: time)
2) Reculer (cost: no reward)
Fallout: Fragment acquired, hint logged.

Example B
Hook: "Un chant sans voix glisse entre les arbres."
Reveal: "Merlin note une trace de pas qui ne revient jamais."
Stakes: Gain a clue, or trigger an anomaly.
Choices:
1) Suivre la trace (cost: risk)
2) Marquer le lieu et partir (cost: no reward)
Fallout: Clue acquired, world tension +1.

Example C
Hook: "Une pierre s'ouvre comme une paupiere."
Reveal: "Un symbole se forme et s'efface aussitot."
Stakes: Gain a lore key, or lose a resource.
Choices:
1) Toucher le symbole (cost: 1 resource)
2) Ignorer (cost: no reward)
Fallout: Lore key acquired, glitch pressure +1.

Event type: Combat node

Example A
Hook: "Le sol se souleve, et la route gronde."
Reveal: "Merlin nomme un Gardien de sel."
Stakes: Reward rune on victory, injury on loss.
Choices:
1) Attaquer vite (cost: energy)
2) Defendre et observer (cost: time)
Fallout: Victory, rune earned.

Example B
Hook: "Une forme traverse la falaise comme un reflet."
Reveal: "Une chasseuse d'ombre bloque le passage."
Stakes: Gain tool, or lose morale.
Choices:
1) Affronter (cost: energy)
2) Contourner (cost: time)
Fallout: Defeat, morale -1.

Example C
Hook: "L'air devient lourd, la bestiole hesite."
Reveal: "Merlin avertit d'un essaim de verre."
Stakes: Victory yields materials, failure yields bond loss.
Choices:
1) Briser l'essaim (cost: energy)
2) Proteger la bestiole (cost: reduced reward)
Fallout: Victory, bond +1.
