# Event Tag Glossary

Use tags to match events with systems and spawns.

Core tags
- ritual: rites, vows, ceremonial actions
- menhir: stone sites, fast travel, access points
- spirit: spirits, ghosts, unseen presence
- brume: fog, reduced clarity, hidden paths
- mystere: puzzles, hidden chains, secret lore
- conflit: conflict pressure and tension

Entities
- korrigan: trickster encounters and misdirection
- raven: omen and inversion events
- guardian: protective entities
- wisp: lights, will o wisps

Activities
- craft: tool creation and upgrades
- resource: harvesting, gathering
- fauna: animal tracking or hunting
- relic: rare artifacts
- trade: exchanges and barter with travelers

Companion / Taniere
- bestiole: companion related event
- taniere: companion den activities
- bestiole_care: care actions (feed, groom, rest)
- taniere_minigame: den mini-games

Audio (SFX)
- sfx_chirp: friendly chirp cue
- sfx_purr: calm purr cue
- sfx_trill: bright trill cue
- sfx_huff: tired/annoyed cue
- sfx_whimper: sad/hurt cue
- sfx_pop: playful pop cue

Temporal
- nuit: night, high mystery pressure
- aube: dawn, promise and source energy
- crepuscule: twilight rituals

System
- hour_bonus: action during hourly bonus
- best_time: action during best time phase
- anomaly: omen or anomaly detected
- guided: guided encounter
- no_hint: mystery solved without hints

Guideline
- Prefer 2-4 tags per event
- Keep tag vocabulary stable
