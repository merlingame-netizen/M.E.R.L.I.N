# Merlin Narrative Templates (Event Types)

Goal
- Stable event types, unique stories each time.
- Merlin narrates with memory and intent while keeping outcomes consistent.

Global rules
- Never repeat the same metaphor in two consecutive events.
- Keep the same event type structure, vary characters, motifs, and cause.
- Mention one memory anchor (from this run) to make it feel coherent.

Template format
- Hook: short, vivid opening line.
- Reveal: the key choice or test is presented.
- Stakes: what is gained or lost, clearly stated.
- Prompt: the choices (2-4) with a visible cost.
- Fallout: short summary after choice resolves.

Event type templates

1) Narrative node
- Hook: "Un signe discret ouvre un passage inattendu..."
- Reveal: "Merlin indique un chemin, mais l'autre voie chuchote un nom."
- Stakes: gain a clue vs lose time.
- Prompt: 2-3 choices.
- Fallout: outcome + new hint.

2) Test node (Force/Logique/Finesse)
- Hook: "Le lieu impose une epreuve."
- Reveal: "Merlin propose un test precis."
- Stakes: success -> reward; failure -> debt or detour.
- Prompt: choose approach or accept the test.
- Fallout: test result and effect.

3) Combat node
- Hook: "Une presence surgit du decor."
- Reveal: "Merlin nomme la menace et ses patterns."
- Stakes: reward vs injury; bestiole bond can shift.
- Prompt: pick tactic or accept combat.
- Fallout: victory/defeat + loot or cost.

4) Discovery node
- Hook: "Un fragment du monde refuse la logique."
- Reveal: anomaly or lore fragment; Merlin hints without revealing.
- Stakes: gain lore key vs risk anomaly.
- Prompt: 2 choices (investigate or bypass).
- Fallout: fragment acquired or pressure increased.

5) Trade node
- Hook: "Un echange s'impose."
- Reveal: trader offers utility tools or runes.
- Stakes: resource cost vs future option.
- Prompt: accept/refuse/bargain.
- Fallout: tool gained or tension rises.

6) Ritual node
- Hook: "Le lieu demande un geste ancien."
- Reveal: a promise hook with a time cost.
- Stakes: immediate gain vs future obligation.
- Prompt: accept/decline or alter the promise.
- Fallout: promise recorded or route shifts.
