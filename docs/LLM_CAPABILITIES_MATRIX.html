Principe Architectural

LLM = peau expressive   |   Code = cerveau logique
Le LLM genere le texte narratif et les choix. Le code GDScript gere la logique (effets, equilibrage, economie, detection). Toute decision mecanique est dans le code. Toute expression creative est dans le LLM.

Limites du 1.5B

Inventaire realiste de ce que Qwen 2.5-1.5B fait bien, moyennement, et ne peut pas faire pour M.E.R.L.I.N.

BIEN (fiable en prod)

  • Generation de texte narratif court (2-4 phrases)
  • Francais courant (80%+ natif)
  • Suivi de format avec GBNF grammar
  • Imitation de style via few-shot
  • JSON simple (3 cles, valeurs numeriques)
  • Reponse a un prompt structure
  • Variete lexicale avec vocabulaire injecte
  • Latence acceptable (7-11s, 17.8 tok/s)

MOYEN (avec tuning)

  • Coherence sur 5+ cartes consecutives
  • Vocabulaire celtique spontane (sans injection)
  • Personnalite stable (derive apres 10 cartes)
  • Adaptation au contexte des etats Triade
  • Creativite des choix (tendance a la repetition)
  • Ton humoristique / 4e mur
  • Sequences de reve (parfois trop generique)

IMPOSSIBLE (ne pas essayer)

  • Raisonnement multi-step (planification)
  • Memoire longue (> 4096 tokens contexte)
  • Auto-correction de ses erreurs
  • JSON complexe / nested objects
  • Calculs numeriques (equilibrage precis)
  • Comprehension de regles de jeu abstraites
  • Generation de code ou de formats techniques
  • Dialogues multi-tours sans reset contexte
