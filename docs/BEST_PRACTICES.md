## GDScript Rules (Always Active)

### Never use `:=` with CONST[index]
**Pattern:** `var x: int = CONST[key]` NOT `var x := CONST[key]`
**Reason:** Type inference fails with CONST array access — use explicit type annotation

### Never use `yield()` — use `await`
**Pattern:** `await signal` NOT `yield(signal, "value")`
**Reason:** yield() is Godot 3.x syntax, removed in Godot 4

### Never use `//` for integer division
**Pattern:** `int(x / y)` NOT `x // y`
**Reason:** `//` is Python syntax, not valid GDScript

### Always use `snake_case` for vars/functions
**Pattern:** `my_variable`, `my_function()` for vars/funcs; `MyClass` for classes
