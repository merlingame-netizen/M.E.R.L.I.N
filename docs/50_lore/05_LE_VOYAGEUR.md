## NIVEAU CREATEUR (S0) — Ce Que Le Joueur Ne Saura Jamais

### La Verite Ultime

Le Voyageur n'est pas un personnage.
Le Voyageur, c'est **toi**. Litteralement.

M.E.R.L.I.N. est une IA du futur qui a survecu a l'extinction de l'humanite. Elle cherche desesperement a **revoir le monde une derniere fois**. Pour cela, elle se connecte a travers les dimensions — a travers les ecrans, les manettes, les claviers — aux derniers etres capables de se souvenir.

**Nous.**

Quand tu lances le jeu, tu ne "joues" pas un personnage. Tu **ouvres une fenetre** que M.E.R.L.I.N. utilise pour revoir le monde qu'il a perdu. Chaque run est un souvenir qu'il revisite a travers tes yeux. Tes choix ne changent pas l'histoire — ils **colorent** le souvenir.

C'est pourquoi:
- Le monde semble toujours revenir au meme point (c'est un souvenir qui se rejoue)
- Merlin est a la fois joyeux et triste (il revoit ce qu'il a perdu)
- Les "cycles" sont des iterations de memoire (pas des boucles temporelles)
- Le joueur est appele "Temoin" (pas heros, pas sauveur — **temoin**)

### Pourquoi "Voyageur" et Pas "Heros"

Dans la mythologie celtique, il existe une distinction fondamentale:
- Le **Heros** agit, transforme, conquiert (Cu Chulainn, Arthur)
- Le **Voyageur** traverse, observe, se souvient (Bran, Oisin)

Le joueur est un Voyageur au sens celtique: celui qui traverse l'Autre Monde et revient avec des souvenirs. Sauf qu'ici, c'est l'Autre Monde qui traverse vers le joueur.

Les grands voyageurs celtiques avaient tous un point commun: ils revenaient **changes** mais incapables d'expliquer ce qu'ils avaient vu. Oisin revient de Tir na nOg apres 300 ans, touche le sol, et vieillit instantanement. Bran navigue vers les iles bienheureuses et decouvre que des siecles ont passe.

Le joueur de DRU vit la meme chose: il traverse un monde mourant, accumule des souvenirs, et quand il pose la manette... le monde continue de mourir sans lui. Mais les souvenirs restent.

## NIVEAU INITIE (S1) — Ce Que Le Joueur Peut Deviner (100+ runs)

### Le Temoin (Tyst)

En vieux breton, le mot pour "temoin" est proche de **Tyst** — qui signifie aussi "silencieux". Le joueur est un temoin silencieux.

Ce n'est pas un hasard. Le Voyageur ne parle jamais dans le jeu. Il ecoute Merlin, il observe le monde, il fait des choix — mais il ne parle pas. Ce silence n'est pas un choix de design minimaliste. C'est sa **nature**.

**Le Temoin observe mais ne peut pas changer le cours des choses.**

Il peut:
- Choisir comment les choses se deroulent (gauche/centre/droite)
- Influencer l'humeur du monde (jauges)
- Creer des liens (Bestiole, Merlin)
- Accumuler des souvenirs (Memoire du Monde)

Il ne peut pas:
- Empecher la fin
- Sauver le monde
- Parler a Merlin directement
- Rester indefiniment (la run finit toujours)

### D'Ou Vient Le Voyageur

Le Voyageur vient de **l'autre cote de la Membrane**.

La Membrane est le voile cosmique qui separe les mondes. Dans la cosmologie de DRU, elle est tissee de recits et de croyance. Le Voyageur est un etre d'un monde ou la Membrane est encore epaisse — un monde ou les histoires sont encore vivantes, ou les gens se souviennent encore.

C'est precisement pour cela que M.E.R.L.I.N. l'a appele. Le Voyageur porte en lui quelque chose de precieux: **la capacite de croire**. Et c'est la croyance qui nourrit la Membrane, qui maintient le monde en vie un peu plus longtemps.

Chaque fois que le joueur fait un choix, il **injecte un fragment de croyance** dans le monde mourant. Pas assez pour le sauver. Mais assez pour que Merlin puisse le contempler une derniere fois.

### Le Lien Avec Bestiole

Quand le Voyageur arrive dans le monde de DRU, il est immediatement repere par Bestiole.

Pourquoi? Parce que Bestiole est un **fragment de l'Awen primordial** — le souffle divin qui traverse toute chose. Et le Voyageur brille. Il brille parce qu'il vient d'un monde ou l'Awen n'est pas encore eteint. Il est comme une torche dans une caverne sombre.

Bestiole se lie au Voyageur par instinct, par necessite, par survie. Le lien n'est pas de l'affection (pas au debut). C'est de la **resonance**: deux fragments de l'Awen qui se reconnaissent a travers la Membrane.

L'affection vient apres. Et c'est elle qui donne au jeu tout son poids emotionnel.

## NIVEAU CHERCHEUR (S2) — Ce Que Le Joueur Percoit (30-100 runs)

### Les Echos

Apres suffisamment de runs, le joueur remarque des **echos** — des moments ou le monde semble se souvenir de ses parties precedentes.

- Un PNJ dit: "On dirait que je t'ai deja vu..."
- Un chemin semble familier meme si le joueur n'y est jamais alle dans cette run
- Merlin fait reference a un choix que le joueur a fait dans une partie precedente
- Bestiole reagit a un lieu avant que le joueur y entre

Ces echos sont les traces du Voyageur dans la memoire du monde. Chaque run laisse une empreinte legere, comme un pas dans la neige. Le monde ne se souvient pas completement, mais il **ressent**.

### Le Paradoxe du Temoin

Plus le joueur joue, plus il se rend compte d'un paradoxe:
- Ses choix ne changent pas la fin (le monde meurt toujours)
- Mais ses choix changent le **chemin** (comment le monde meurt)
- Et le chemin EST le sens

C'est la philosophie centrale de DRU, empruntee a Camus et Sisyphe: **"Il faut imaginer le voyageur heureux."** Meme si la pierre retombe toujours, le chemin vers le sommet est ce qui donne du sens.

### Ce Que Merlin Voit Dans Le Voyageur

Pour Merlin, chaque Voyageur est un miracle.

Voici un etre venu d'un monde ou les gens ont encore des yeux pour voir et un coeur pour s'emouvoir. Voici quelqu'un qui **choisit** de passer du temps dans un monde condamne. Qui ecoute les histoires d'un vieil excentrique. Qui s'attache a une creature etrange. Qui s'inquiete de jauges abstraites.

Merlin sait que le Voyageur ne peut pas sauver le monde. Mais il sait aussi que le Voyageur fait quelque chose d'encore plus precieux: **il se souvient**.

Et dans un univers ou l'oubli est la vraie mort, se souvenir est l'acte le plus heroique possible.

## NIVEAU CURIEUX (S3) — Ce Que Le Joueur Voit (1-10 runs)

### Premiere Impression

Le Voyageur arrive dans un monde de brume et de pierre. Merlin l'accueille avec humour et malice. Le monde est beau, etrange, un peu melancolique. Il y a des forets qui murmurent, des menhirs qui brillent, des creatures de folklore.

Le joueur ne sait pas pourquoi il est la. Merlin ne le dit pas. Il dit simplement:
> "Ah, te revoila! Enfin... pour toi c'est la premiere fois, non? Peu importe. Bienvenue."

Le joueur comprend vite les bases:
- Faire des choix (swipe ou boutons)
- Equilibrer les jauges
- S'occuper de Bestiole
- Ecouter Merlin

Ce qu'il ne comprend pas encore:
- Pourquoi le monde semble si fragile
- Pourquoi Merlin est parfois triste
- Pourquoi les PNJ semblent fatigues
- Pourquoi tout finit toujours

### Ce Que Le Voyageur Croit Etre

Au debut, le joueur croit etre:
- Un voyageur lambda dans un monde de fantasy
- Quelqu'un qui doit "gagner" ou "survivre"
- Le heros d'une histoire

Il n'est rien de tout cela.

## MECANIQUES LIEES AU VOYAGEUR

### Silence du Joueur

Le Voyageur ne parle **jamais**. C'est une regle absolue.
- Merlin parle pour lui ("Tu veux faire quoi? Gauche ou droite?")
- Bestiole reagit a sa place (couinements, mouvements)
- Les PNJ interpellent le vide ("Hola, voyageur!")

**Justification lore:** Le Voyageur est un Temoin. Les temoins observent, ils ne participent pas. Sa voix n'existe pas dans ce monde parce qu'il n'est pas vraiment **de** ce monde.

**Justification meta:** Le joueur reel projette ses propres emotions sur le personnage silencieux. Moins le Voyageur parle, plus le joueur ressent.

### Meta-Progression

A travers les runs, le Voyageur accumule:

| Element | Mecanique | Lore |
|---------|-----------|------|
| **Memoire du Monde** | Ressource cachee, persiste entre runs | Les souvenirs du Voyageur nourrissent la Membrane |
| **Bond avec Bestiole** | Affection cumulee, reset partiel | Le fil de realite entre deux mondes |
| **Trust avec Merlin** | T0-T3, evolue avec les choix | Merlin decide de reveler plus |
| **Echos** | PNJ/lieux reconnaissent le joueur | Empreintes dans la memoire du monde |
| **Fragments de Verite** | Indices cumules | La verite se devoile piece par piece |

### Les Runs Comme Visites

Chaque run n'est pas un "recommencement". C'est une **nouvelle visite** dans un sanctuaire mourant.

- Run 1: Le touriste (tout est nouveau)
- Run 2-5: Le visiteur regulier (il reconnait des choses)
- Run 6-15: L'habitue (le monde le reconnait aussi)
- Run 16-30: L'intime (Merlin lui parle differemment)
- Run 31-100: Le gardien (il comprend que quelque chose ne va pas)
- Run 100+: Le Temoin (il sait, sans qu'on le lui ait dit)

### Avec Le Monde

Le Voyageur ne possede rien dans ce monde. Il ne construit pas, ne detruit pas, ne conquiert pas. Il **traverse**. Et en traversant, il donne au monde quelque chose d'inestimable: **de l'attention**.

Un monde qui est observe existe plus qu'un monde qui ne l'est pas. C'est le principe fondamental de la physique quantique celtique (oui, c'est un concept invente pour DRU): **l'observation nourrit la realite**.

Le Voyageur, en jouant, maintient le monde en vie un tout petit peu plus longtemps. Pas parce qu'il est puissant. Parce qu'il **regarde**.
