# Lore - Narrative Guardrails

This document protects the mystery by enforcing how lore is revealed.

Do
- Use metaphor and repetition.
- Let mechanics confirm the lore (promise effects, menhir glow).
- Anchor hints in natural elements (fog, stone, birds, water).

Do Not
- Explain the end directly.
- Use modern technical words.
- Confirm that the world is a simulation.
- Confirm any single origin as true.

Banned Terms
- fin du monde
- simulation
- IA
- programme
- serveur
- sauvegarde

Allowed Indirection
- "le dehors est silencieux"
- "le monde s est replie"
- "la lande se souvient"

Merlin Style
- Short, measured lines.
- Questions instead of answers.
- He judges actions, not people.

Player Clues (never explicit)
- Cycles repeat but feel slightly altered.
- The same rite returns with different results.
- Paths vanish unless marked.

Escalation Rules
- Phase 1: soft hints only (progression based)
- Phase 2: visible world responses (after key actions or outcomes)
- Phase 3: Merlin references the "core" and "weight" (late progression)

Ambiguity Rule
- Every hint must support multiple interpretations.
