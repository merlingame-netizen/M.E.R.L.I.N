# Lore Bible - M.E.R.L.I.N (Hints-Only)

Purpose
- Define the lore direction without explicit reveals.
- Provide consistent hint language and boundaries.

Non-Truth Rule
- No statement in this doc is a confirmed fact.
- Everything is phrased as signs, rumors, or patterns.

Foundations (interpretations)
- The land feels folded, as if it is holding itself together.
- Cycles feel intentional, as if keeping balance.
- Menhirs feel more like anchors than monuments.
- Fog behaves like a veil that hides fragile seams.
- Promises feel structural, not only moral.

M.E.R.L.I.N (folk readings)
- Many assign meanings to the letters:
  - Memoire, Equilibre, Rites, Lieux, Influx, Noyau
- No one agrees if this is true, but the words fit the land.

Merlin (tone and role)
- He tests actions rather than explaining causes.
- He speaks in short, poetic lines.
- He never confirms a rumor.

Promise Logic (felt consequences)
- Truth opens paths; lies narrow them.
- The land responds to vows more than to force.
- This is shown, not told.

Menhir Logic (observed)
- Menhirs react to resolved vows and strong balance.
- Pathways appear when tension falls.

Fog Logic (observed)
- Fog thickens after broken promises.
- Fog lifts after honest acts.

Never Stated Directly
- No explicit origin of the land.
- No explicit definition of M.E.R.L.I.N.
- No explicit role for the player.

Language and Tone Rules
- Use metaphor and repetition.
- Avoid modern technical words.
- Prefer nature imagery (stone, water, birds, wind).

Reveal Arc (progression only)
- Phase 1: soft hints only
- Phase 2: visible world responses
- Phase 3: Merlin mentions "core" and "weight" without explaining

Qualification Notes (for design)
- Hints must fit mechanics (promises, fog, menhirs).
- Calendar events remain bonus only; no lore gating.
- Keep a cap: one hint per day at most.
