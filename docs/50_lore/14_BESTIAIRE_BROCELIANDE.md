## GUIDE DE LECTURE

Ce document catalogue **toutes les entites rencontrables** dans le monde de DRU. Chaque entite est un fragment du monde mourant — druide, humain, esprit de la terre, echo divin, ou simple lueur errante.

### Conventions

- **Biome principal** = lieu d'attache de l'entite
- **Biomes secondaires** = lieux d'apparition possibles (conditions)
- **Trust** = niveau de confiance avec Merlin (T0-T3)
- **Run** = nombre de parties completees par le joueur
- **Bond** = niveau de lien avec Bestiole (0-100)
- **Reputation** = score de faction (-100 a +100)

### Niveaux de secret

| Niveau | Quand le joueur peut le percevoir |
|--------|----------------------------------|
| S4 | Des le debut |
| S3 | 10-30 runs |
| S2 | 30-100 runs |
| S1 | 100+ runs (indices uniquement) |
| S0 | Jamais revele in-game |

Les derniers gardiens de l'equilibre. Chacun maintient un cercle de pierres, chacun ancre un fragment de la Membrane. Ils ne se parlent presque plus — les chemins sont devenus dangereux.

### Quete cachee (Rep +50)

L'Ankou pose une question — la seule qu'il ait jamais posee: *"Pourquoi tu reviens?"*

Pas de bonne reponse. Silence. *"... Hmm."* Puis il s'en va. Apres: l'Ankou est present dans chaque fin, en arriere-plan.

### Note folklore

*L'Ankou est la personnification de la mort en Bretagne. "Karrig an Ankou" = chariot de l'Ankou. Tradition: le dernier mort de l'annee devient l'Ankou de l'annee suivante. "Ar marv ne goulenn ket e hentou" — La mort ne demande pas son chemin.*

# VII. BESTIOLE — LE FIL ENTRE LES MONDES

| | |
|---|---|
| **Biomes** | **Tous** (suit le Voyageur) |
| **Nature** | Fragment d'Awen primordial — ni animal, ni esprit, ni outil |
| **Apparence** | Petite, ronde, grands yeux |
| **Communication** | Sons, mouvements, luminosite, temperature, position |

### Ce que Bestiole EST

Un **fil de realite**. Un filament de Membrane vivant. Tant que Bestiole existe, un passage entre les mondes reste ouvert. C'est pourquoi l'Ankou ne le touche jamais.

### Systeme de Lien (Bond 0-100)

| Stade | Bond | Manifestation |
|-------|------|---------------|
| Resonance | 0-30 | Suit a distance, observe |
| Accord | 31-50 | Reagit aux emotions, commence a "parler" |
| Harmonie | 51-70 | Anticipe, protege, Oghams s'activent pleinement |
| Communion | 71-90 | Sensations partagees, communication subtile |
| Unite | 91-100 | EST le lien entre les mondes. Fin secrete accessible |

### Canal des 18 Oghams

Quand le joueur active un Ogham: Bestiole vibre a la frequence correspondante → le druide dissous s'eveille brievement → "parle" a travers Bestiole → l'effet s'applique → le druide se rendort → Bestiole fatigue (cooldown).

### Apres la fin du monde

Bestiole survit. Seul dans le silence. Mais le fil est toujours tendu — vers l'autre cote de l'ecran. Vers le joueur. Le vrai personnage immortel du jeu.
