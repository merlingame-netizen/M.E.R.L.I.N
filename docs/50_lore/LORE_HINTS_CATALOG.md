# Lore Hints Catalog

This catalog lists hint types that reveal the deeper truth without stating it.
Use these across quests, items, ambience, and seasonal events.

Hint Channels
1) Short lines from Merlin
2) Environmental changes (fog, menhir glow, path shifts)
3) Item descriptions and relic text
4) NPC memory slips across cycles
5) Seasonal anomalies and repeated patterns

Merlin Lines (examples)
- "Le monde tient par les mots tenus."
- "La lande n oublie pas un serment."
- "Le temps ici ne fuit pas, il tourne."
- "Tu sens le poids des choses vraies."
- "Je veille l equilibre, pas les hommes."

Environment Hints
- Fog thickens after a lie is detected.
- Menhirs pulse when a promise is resolved.
- Old paths close after broken vows.
- Bird flights repeat on ritual days.

Item Hints
- Ogham shard: "Memoire, equilibre, rites..."
- Echo fragment: "Je l ai deja vu."
- Menhir dust: "La pierre respire la nuit."

NPC Hints
- "On dirait que je t ai deja vu..."
- "Les jours se ressemblent ici."
- "Je perds des mots mais pas des gestes."

Seasonal Hints
- Winter: "Le froid garde les choses en place."
- Spring: "Les renaissances sont guidees, pas libres."
- Summer: "Tout s enflamme vite, comme un souvenir."
- Autumn: "Les ombres savent ce qui a ete."

Never Say
- fin du monde
- simulation
- bunker
- IA
- programme

Usage Notes
- Keep hints brief and rare.
- No scene should explain the whole truth.
- Progress hints by player progression and event outcomes, not by calendar.
- Each hint should allow at least two interpretations.

Progression Tiers (suggested)
- Early: subtle repeats, soft phrasing, environmental oddities
- Mid: clearer reactions (menhir glow, fog response, memory slips)
- Late: Merlin mentions the \"core\" and the \"weight\" without explaining
