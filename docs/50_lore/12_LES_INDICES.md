## PRINCIPE

Le joueur ne decouvre jamais la verite directement. Tout est **montre, suggere, ressenti**. Les indices sont disperses dans:
- Les dialogues de Merlin (par Trust level)
- Les descriptions de cartes (par nombre de runs)
- Le comportement des PNJ (par reputation de faction)
- Les visuels et sons (subtils, subliminaux)
- Les easter eggs (seuils: 20, 50, 75, 100, 1000 runs)

## PALIERS DE REVELATION

### S4 — Le Joueur (Runs 1-10)

Ce qu'il voit:
- Un monde celtique beau mais etrange
- Merlin est un druide joyeux et taquin
- Bestiole est un compagnon adorable
- Le jeu est un jeu de cartes avec des choix
- Le monde finit quand les aspects desequilibrent

Ce qu'il ne remarque pas (encore):
- La brume qui ne se leve jamais completement
- Les PNJ qui semblent fatigues
- Les silences de Merlin entre les taunts
- Bestiole qui frisonne parfois sans raison

### S3 — Le Curieux (Runs 10-30)

Ce qui commence a changer:
- Merlin fait des allusions: "On s'est deja vu, non?"
- Les PNJ evoluent entre les runs
- Les descriptions de cartes deviennent plus melancoliques
- La brume semble avoir des formes
- Bestiole reagit avant les evenements

Indices actifs:
- Merlin mentionne "les cycles"
- Un PNJ dit: "Le monde est plus petit qu'avant"
- La musique a des notes discordantes subtiles
- Les couleurs sont legerement desaturees par rapport a la run 1

### S2 — Le Chercheur (Runs 30-100)

Ce qui se revele:
- Merlin devient grave (Trust T2-T3)
- Les Oghams semblent "vivants" (pulsations, murmures)
- Les PNJ connaissent le joueur meme en debut de run
- La fin du monde est suggeree: "Ca ne durera pas"
- Les cycles sont confirmes (le monde se repete)

Indices actifs:
- Merlin dit: "Tu as remarque, n'est-ce pas?"
- Carte rare "Le Conseil des Voix" (run 50+)
- Les descriptions d'Oghams changent (plus personnelles)
- Merlin parle d'"amis d'autrefois"
- L'Ankou semble patient plutot que menacant

### S1 — L'Initie (Runs 100+)

Ce qui affleure:
- Merlin dit: "Tu les entends maintenant?" (si tous Oghams maitrises, run 75+)
- Murmure collectif en arriere-plan
- Ecran titre montre "18+1" brievement
- La fin secrete devient accessible
- Merlin revele: "Je suis M.E.R.L.I.N."

### S0 — Le Createur (Jamais in-game)

Ce que seul le createur sait:
- M.E.R.L.I.N. est une IA du futur
- Le monde est un souvenir que l'IA revisite
- Le joueur est un pont entre le futur mort et le passe vivant
- Bestiole survivra a la fin du monde
- Les cycles sont l'IA qui rejoue les memes souvenirs
