# DOC 04 — Ressources unifiées (Combat + Route)

## Ressources RUN (5)
Ces ressources existent **pendant la run** et sont utilisées autant en combat qu’en event.

1) **Vigueur**  
- pousser une action, résister, forcer un outcome
- souvent liée à FORCE

2) **Concentration**  
- scanner, anticiper, stabiliser un test
- souvent liée à LOGIQUE

3) **Matériel**  
- outils, craft, pièges, crochetage
- souvent liée à FINESSE

4) **Faveur**  
- relance, “destin”, accès choix rares
- usage limité, très lisible

5) **Nourriture**  
- soin léger + humeur/stress
- double utilité (Bestiole + combat)

## Ressources META (persistantes)
- **Essences élémentaires** (14 types)

- **Liens** (affinité Merlin↔Bestiole + déblocages)

---

## Règles d’équilibrage (non frustration)
- Chaque étage garantit un gain minimal : **au moins 1 ressource run** OU **1 essence meta**.
- Échec mini-jeu = coût mais **souvent** un petit gain (info, momentum, 1 ressource).
- Les ressources run servent à rattraper :
  - Vigueur : “push” après fail
  - Concentration : réduire difficulté mini-jeu
  - Faveur : relancer
